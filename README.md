📱 Menu Responsivo

Projeto de Menu Responsivo desenvolvido utilizando HTML, CSS e JavaScript, com foco em criar uma navegação moderna e adaptável para diferentes tamanhos de tela.

O menu se adapta automaticamente para dispositivos móveis e desktops, utilizando um botão de hamburger menu que abre e fecha a navegação de forma dinâmica.

Este projeto é ideal para praticar conceitos de responsividade, manipulação do DOM e interatividade em interfaces web.

🚀 Tecnologias Utilizadas

HTML5 – Estrutura da página

CSS3 – Estilização e responsividade

JavaScript (Vanilla JS) – Controle de abertura e fechamento do menu

SVG – Ícones de menu e fechamento

🎯 Funcionalidades

✔️ Menu adaptável para diferentes tamanhos de tela
✔️ Botão Hamburger Menu para dispositivos móveis
✔️ Animação de abertura e fechamento do menu
✔️ Ícones SVG para melhor qualidade visual
✔️ Estrutura simples e reutilizável em outros projetos
